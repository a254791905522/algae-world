# Algae World Release Package

App name: Algae World

Bundle ID: com.rosewood.algaeworld.game

Device support: iPhone only

Gameplay fact card:
- Core action: drag the lamp position around the tank.
- Main puzzle: keep algae in usable light while avoiding too dark or over-lit states.
- Nutrient base: N, P, and K capsules support growth for different algae.
- Progression: 50 short levels introduce green algae, cyanobacteria, red algae, golden algae, oxygen pressure, and compound orders.
- Monetization and network: offline, no ads, no in-app purchases, no account, no analytics, no tracking.
